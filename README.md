#WP Gallery Zplus

WordPress Gallery plugin allows you to create an entire image gallery on your website that showcases your portfolio or product view from different aspects. You can create albums to categories the images, and upload multiple images at once that save your time. With this Gallery plugin for WordPress, you can add the code to display the gallery on any specific page.

#Prerequisites

Wordpress must be installed, atleast 4.2.0

#Installation

1. Go to Plugins -> Add New.
2. Click "Upload" right underneath "Install Plugins".
3. Upload the zip file that this readme was contained in.
4. Activate the plugin.
5. Go to Gallery ZPlus -> Manage Galleries 
6. Add galleries, images and setting etc.
